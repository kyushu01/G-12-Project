Tic-Tac-Toe Game

Game Rules

  1. This is a two-player game. One player is "X", and the other is "O".
  2. Players take turns dragging their symbols onto the game board.
  3. The first player to align three symbols horizontally, vertically, or diagonally wins the game.
  4. If all cells are filled and no one has won, the game is a draw.
  5. Click the "Reset Game" button to start a new game at any time.
